# Authorship And Provenance

This file records which material is original human-authored work and which material was generated or derived later.

## Default Rule

All existing notebooks in this repository are treated as original human-authored notes by:

```text
B67687
```

unless a specific file says otherwise.

This includes the original Jupyter notebooks under:

```text
algebra/
calculus/
discrete-mathematics/
fractals/
trigonometry/
translated-notebooks/
```

## Converted Markdown (Archived)

The markdown-canonical strategy has been superseded (see `markdown-migration.md`).
Derived Markdown files from the notebook→markdown conversion are now archived under:

```text
archive/markdown/
```

They retain their original front matter for provenance but are no longer canonical.
The Jupyter notebooks in the active directories are the canonical source format.

## Generated Or Derived Notes

New explanations created from prompts, plans, or later derivation should use:

```yaml
generated_by: GPT-5
review_status: needs-human-review
```

If the note is later reviewed and rewritten manually, add:

```yaml
human_reviewed_by: B67687
review_status: human-reviewed
```
